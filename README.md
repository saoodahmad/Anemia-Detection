# Anemia Detection Using DL

## Prerequisite

- Install Python 3.10.12

- Install Node 20.11.0

- Install libgl1 on Ubuntu using apt-get

- Unzip images.zip to get the images required for prediction

## Project Setup

### Backend

```
    cd Backend
    pip install -r requirements.txt
    python3 index.py
```

### UI

```
    cd ui
    npm install
    npm run dev
```
